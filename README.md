## 🚀 How to execute the project
#### Clone the repository
git clone https://github.com/EPieritz/radio-podcast.git

#### Enter directory
`cd radio-podcast`

#### Download dependencies
`npm install`

#### Run the server
`npm start`

#### That done, open your browser and go to `https://localhost:3000/`
